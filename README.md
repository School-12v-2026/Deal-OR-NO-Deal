# Deal-OR-NO-Deal
Deal or No Deal е конзолна игра, вдъхновена от популярното телевизионно предаване. Играчът избира една кутия и постепенно отваря останалите, като следи кои парични суми отпадат. След всеки рунд Банкерът прави оферта, а играчът трябва да вземе решение дали да я приеме или да продължи играта.
